# Port Scanner with Service Detection

A Python-based network reconnaissance tool that scans target hosts for open ports and identifies running services.

## Features
- Scans common ports (21, 22, 23, 25, 53, 80, 110, 143, 443, 445, 3306, 3389, 8080)
- Custom port range scanning capability
- Service identification (SSH, HTTP, FTP, MySQL, RDP, etc.)
- Comprehensive error handling

## Technical Skills Demonstrated
- Socket programming in Python
- TCP/IP protocol understanding
- Network security reconnaissance
- Exception handling and input validation

## Usage
```bash
python3 port_scanner.py
```

Enter target IP or hostname when prompted. Choose whether to scan custom port range.

For educational purposes only. Only scan systems you have permission to test.
