# To-Do List Application with File Persistence

A command-line to-do list application with full CRUD operations and file-based data persistence.

## Features
- Add, view, and remove tasks
- Data persists across sessions using file I/O
- Simple menu-driven interface
- Task management with numbered list

## Technical Skills Demonstrated
- File handling in Python (reading/writing)
- CRUD operations (Create, Read, Update, Delete)
- Function-based code organization
- User input validation
- Data persistence concepts

## Usage
```bash
python3 todo_app.py
```

Choose options from menu:
1. View tasks
2. Add task
3. Remove task
4. Exit
