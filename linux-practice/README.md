# System Monitoring Script

Bash script for comprehensive system health monitoring and reporting.

## Features
- Displays system uptime and current date/time
- Reports memory usage (free/used)
- Shows disk usage for mounted filesystems
- Lists top CPU-consuming processes
- Shows currently logged-in users
- Generates formatted report

## Technical Skills Demonstrated
- Bash scripting
- Linux system administration
- Command-line tools (uptime, free, df, ps, who)
- Output formatting and report generation
- System monitoring concepts

## Usage
```bash
chmod +x system_monitor.sh
./system_monitor.sh
```
## Use Cases
- Regular system health checks
- Monitoring server performance
- Identifying resource usage issues
- Security auditing (tracking logged-in users)
